El proyecto AccessiLearn es una aplicación móvil diseñada para apoyar a estudiantes universitarios con discapacidad visual. Su objetivo principal es facilitar su vida académica mediante herramientas accesibles como gestión de horarios, lectura de documentos con voz (OCR), navegación dentro del campus y notificaciones inteligentes. Todo está pensado para promover la autonomía e inclusión digital, cumpliendo con altos estándares de accesibilidad.
