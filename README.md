📱 AccessiLearn

Aplicación móvil inclusiva para estudiantes con discapacidad visual

Proyecto académico enfocado en el desarrollo de una solución tecnológica accesible que permite a estudiantes universitarios con discapacidad visual gestionar su vida académica de manera autónoma. La aplicación integra herramientas de lectura de texto, organización de horarios y navegación asistida mediante una interfaz accesible.

🖥️ Vista de la aplicación

Pantalla principal
Módulo de lectura OCR
Gestión de horarios
Navegación en campus

📊 Descripción del proyecto

Los estudiantes con discapacidad visual enfrentan múltiples barreras en el entorno universitario, especialmente en el acceso a materiales académicos, organización del tiempo y desplazamiento dentro del campus.

Para abordar esta problemática, se desarrolló AccessiLearn, una aplicación móvil que centraliza funcionalidades clave en una sola plataforma accesible. La solución incorpora tecnologías de reconocimiento de texto, lectura por voz y navegación asistida, permitiendo transformar información visual en formatos auditivos y mejorar la interacción con el entorno académico.
