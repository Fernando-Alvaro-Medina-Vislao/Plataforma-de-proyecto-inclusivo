# 📱 AccessiLearn - Aplicación Inclusiva para Estudiantes con Discapacidad Visual

## 📌 Descripción del Proyecto

AccessiLearn es una aplicación móvil diseñada para apoyar a estudiantes universitarios con discapacidad visual, facilitando su acceso a información académica y mejorando su autonomía dentro del entorno educativo mediante herramientas accesibles.

---

## 🖥️ Vista de la Aplicación

* Pantalla principal
* Módulo de lectura de texto (OCR)
* Gestión de horarios
* Navegación dentro del campus
* Sistema de notificaciones

---

## 🎯 Funcionalidades Principales

* 🔊 Lectura de documentos mediante OCR
* 🗣️ Conversión de texto a voz (Text-to-Speech)
* 🗓️ Gestión de horarios académicos
* 🔔 Notificaciones y recordatorios
* 🧭 Navegación asistida en campus
* ♿ Interfaz accesible

---

## 📊 Componentes del Sistema
